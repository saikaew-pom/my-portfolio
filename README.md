# Professional Portfolio - Watcharapong (Pom) Saikaew

A modern, high-end professional portfolio website for **Watcharapong Saikaew**, a Director of Events and Hospitality Leader with over 17 years of experience in luxury hotels and destination resorts.

## 🚀 Project Overview

This portfolio is designed to showcase expertise in:
- **MICE & Corporate Congresses**: Large-scale logistics and stakeholder management.
- **Incentive Travel Programs**: Bespoke destination experiences.
- **Destination & Indian Weddings**: High-end, multi-day luxury celebrations.
- **Hotel Buyouts & Special Events**: Full-hotel operational coordination.

## ✨ Key Features

- **Modern & Classy Dark Theme**: A minimalist, high-contrast design using charcoal grays and elegant accents.
- **Interactive Work Gallery**: Showcasing a curated selection of events planned and executed across Thailand.
- **Professional Timeline**: A detailed career progression from coordinator to Director of Events.
- **Credentials & Certifications**: Highlighted international certifications including CMP, CIS, CEM, SEP, and DES.
- **Responsive Design**: Fully optimized for all devices (mobile, tablet, and desktop).
- **Downloadable Resume**: Easy access to the latest PDF CV.

## 🛠️ Technical Stack

- **HTML5**: Semantic structure.
- **CSS3 (Vanilla)**: Custom styling with CSS variables, Flexbox, and Grid. No external frameworks for maximum performance.
- **JavaScript (ES6+)**: Smooth scrolling, reveal animations, and interactive navigation.
- **Typography**: Elegant "Cormorant Garamond" from Google Fonts.
